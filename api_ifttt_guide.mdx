## BibiGPT API概览

- **获取API Token**
- 在BibiGPT个人中心的「笔记集成」处生成API Token链接。

- **API功能**
- 输入视频URL，如B站或YouTube链接，API提供视频摘要与详细信息。

- **测试与使用**
- 使用Postman等工具测试API，并且在移动端结合iOS快捷指令使用。

- **数据安全**
- 遭到泄露时通过个人中心重置API Token保障安全。

## IFTTT智能自动化实例

- **IFTTT简介**
- IFTTT（If This Then That）是一款可以创建自动化任务的工具。

- **操作步骤**

1. 登录IFTTT平台创建新的自动化任务。
2. 设定触发条件，如新视频发布或日历事件。
3. 将BibiGPT API作为动作，使用API Token进行配置。
4. 保存并启用任务，实现自动化操作如视频摘要保存至指定位置。

## 3. 移动端应用集成

- **快捷指令集成**
- 在iOS快捷指令中配置与BibiGPT API对接的动作。
- 将IFTTT自动化配方与快捷指令联动。

## 4. API与IFTTT扩展应用

- **工具集成**
- 利用API实现与其他应用如Obsidian、Roam Research等联动。

- **自动化示例**
- 如Up主发布新视频后自动进行摘要并发送至邮箱。
